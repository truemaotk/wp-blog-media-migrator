在 WordPress 网站之间迁移博客文章、正文、分类标签、特色图、正文图片和阅读量。

== 使用方法 ==

1. 在旧站安装并启用插件。
2. 进入“工具 → 博客文章与图片迁移”，下载文章迁移包。
3. 在新站安装并启用同一个插件。
4. 上传迁移 ZIP，选择是否覆盖同别名文章，然后开始导入。
5. 导入前建议备份新站数据库和 wp-content/uploads。

== 迁移内容 ==

* 标题、别名、正文、摘要、发布日期、状态和文章密码。
* 分类、标签、特色图片。
* 正文中的 src、srcset、data-src、data-original 和 data-lazy-src 图片。
* 常见阅读量字段，以及名称包含 view、read 或 hit 的数字型文章字段。

== 不迁移 ==

* 用户账号、评论、页面、商品、插件和主题设置。
* 无法从旧站访问或被防盗链拦截的远程图片。

== 文章状态说明 ==

* 默认只选择“已发布”，不会导出私密、草稿、待审或定时文章。
* 需要迁移私密文章时，请在导出页面单独勾选“私密”。
* 导入时勾选“保留原文章状态”，私密文章在新站仍然保持私密。
* 如果取消“保留原文章状态”，所有文章都会先保存为草稿。
